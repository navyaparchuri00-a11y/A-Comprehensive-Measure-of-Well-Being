# HDI Prediction System - A Comprehensive Measure of Well-Being

## 📋 Project Overview
This web application predicts the Human Development Index (HDI) category of countries based on key indicators: Life Expectancy, Expected Years of Schooling, Mean Years of Schooling, and Gross National Income (GNI) per capita.

## 🌟 Features
- **User Authentication**: Secure login system
- **HDI Prediction**: Machine Learning-based prediction of HDI tiers (Very High, High, Medium, Low)
- **Prediction History**: Track all your past predictions
- **Interactive Dashboard**: User-friendly interface
- **Data Visualization**: Visual representation of HDI indicators
